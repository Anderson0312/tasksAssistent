**tasksAssistent**

Este repositório contém um assistente de tarefas simples, desenvolvido para ajudar na organização e acompanhamento de tarefas diárias. Se você está procurando uma maneira fácil e rápida de gerenciar suas tarefas, este é o lugar certo!

### Funcionalidades

* Adicionar novas tarefas com descrição e data de vencimento.
* Visualizar todas as tarefas pendentes.
* Marcar tarefas como concluídas.
* Excluir tarefas que não são mais necessárias.
